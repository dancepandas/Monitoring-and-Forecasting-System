# 水文监测指挥核心 — 使用简介

## 系统概况

本系统是一套面向流域水文监测的指挥调度平台，实时采集水位、流量数据，集成 Chronos 时序预报模型和 FloodMind 智能助手，支持预警研判、日报生成和系统自检。

数据来源：aiflow2 水文数据平台（每 5 分钟自动同步）。当前覆盖三个测站：仙桃站（00106）、城西站（00107）、南桥站（00108）。

---

## 页面导航

左侧边栏提供 7 个功能入口，徽章数字实时更新：

| 入口 | 功能 |
|------|------|
| 流域总览 | 首页仪表盘，一站看清全部核心数据 |
| 监测站点 | 各站设备状态与水位流量详情 |
| 预警处置 | 预警与告警列表，附带处置建议 |
| 模型预报 | 配置参数运行时序预报模型 |
| 视频巡检 | 各站视频链路状态 |
| 日报归档 | 查看、预览、导出已生成的日报 |
| 智能体对话 | FloodMind 智能助手交互区 |

---

## 首页（流域总览）

**顶部四卡片**：最新水位、最新流量、模型可信度、待处置预警数。数据每 60 秒自动刷新。

**中间三面板**：
- 视频巡检 — 各站摄像头状态
- 监测设备分布与流域态势 — 流域 SVG 地图，可切换图层（流域态势/风险热力/设备链路/调度推演）
- 预警与告警 — 按紧急程度（红→橙→黄→蓝→提示）排序的实时列表

**底部长图**：历史实测流量 + 未来 12h 预报曲线（蓝色实线 = 实测，橙色虚线 = 预报）

**右键任意面板** → 「询问智能体」→ 自动将面板数据发送给 FloodMind 分析。

---

## FloodMind 智能体

### 入口方式

1. 点击首页右上角「智能研判」按钮
2. 左侧导航进入「智能体对话」页
3. 右键任意面板 →「询问智能体」

### 常用任务快捷按钮

智能体页面提供 8 个一键任务：

| 任务 | 功能 |
|------|------|
| 最新水情 | 查询最新水位和设备状态 |
| 趋势分析 | 分析近期水位/流量变化趋势 |
| 流量预报 | 运行模型预测未来 12h 流量 |
| 预警研判 | 检查预警状态并给出处置建议 |
| 生成日报 | 生成昨日水情日报 |
| 多站对比 | 对比三个站点水情数据 |
| 设备巡检 | 检查设备在线状态与视频监控 |
| 定时日报 | 创建每天 0 点自动生成日报的定时任务 |

### 智能体能做什么

- **数据查询**：水位、流量、设备状态、视频链路、预警信息
- **数据分析**：趋势判断、多站对比、异常检测
- **预报模型**：调用 Chronos 运行时序预测
- **报告生成**：自动生成日报/周报，保存为文件
- **系统诊断**：检查缓存状态、数据时效，判断是单站故障还是全局故障
- **定时任务**：设置每天/每周自动运行的日报、周报、系统检查

## 模型预报页

**参数配置**：
- 预测步长：1–168 小时（首页默认 12h，此页可调更长）
- 上下文窗口：模型输入的历史数据点数
- 置信区间：5%/10%/20%/25% 可选
- 模式：univariate / multivariate / hybrid / ensemble

点击「运行预报」，下方展示历史序列 + 预测曲线，以及峰值、峰现时间、置信上下限。

---

## 日报归档页

**日报列表**：点击任意类型（如「流域运行日报」）→ 弹窗展示最新内容，可「导出 Word」。

**归档统计**：点击统计项 → 弹窗按日期倒序列出所有历史报告 → 点击任意一份查看内容 → 导出 Word。

智能体可设置每天 0:00 自动生成日报、每周一自动生成周报。

---

## 系统状态

顶部栏右上角的胶囊状状态指示器，实时反映系统健康度：

- 绿色「系统正常」— 所有站点数据可用
- 黄色「部分异常」— 部分站点数据延迟
- 红色「数据异常/系统故障」— 需要排查

点击可手动触发系统自检，每 60 秒自动更新。

---

## 预警告警体系

系统有两类提醒：

**预警**（提前防范）：水位超限、流量超限、水位变化率异常、多站联动预警。按蓝/黄/橙/红分级。

**告警**（已发生，需处理）：数据缓存异常、数据停更、数据缺测、数据跳变、采集程序异常、全站不可用、日报生成失败、AI 服务不可达。

所有消息以测站名称（非编码）展示，系统问题附带管理员联系方式。

---

## 快速上手流程

1. 打开首页 → 看顶部四卡片确认当前水情正常
2. 看一眼右上角系统状态 → 确认绿色
3. 有事想问 → 右键任何面板选「询问智能体」或点击「智能研判」
4. 想生成日报 → 进入智能体对话页，点「生成日报」
5. 想设自动日报 → 智能体里点「定时日报」
6. 想看详细预报 → 进入「模型预报」页，设参数后运行
